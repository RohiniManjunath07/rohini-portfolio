## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/RohiniManjunath07/rohini-portfolio.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
